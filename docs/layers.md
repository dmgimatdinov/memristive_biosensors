

# Характеристики слоев биосенсора

## 3.1 Целевой аналит (Target Analyte)

Описывает свойства целевого аналита — вещества, которое должен детектировать биосенсор.

**Таблица в БД:** `Analytes`


| Обозначение | Название характеристики | Тип данных | Единицы измерения | Диапазон значений | Описание |
| :-- | :-- | :-- | :-- | :-- | :-- |
| `TA_ID` | Идентификатор аналита | VARCHAR | — | — | Уникальный ID (например, TA001) |
| `TA_Name` | Название аналита | VARCHAR | — | — | Полное название целевого аналита |
| `PH_Min` | pH минимум | REAL | единицы pH | 2.0 – 10.0 | Минимальное значение pH для стабильности аналита |
| `PH_Max` | pH максимум | REAL | единицы pH | 2.0 – 10.0 | Максимальное значение pH для стабильности аналита |
| `T_Max` | Максимальная температура | REAL | °C | 0 – 180 | Максимальная рабочая температура |
| `ST` | Стабильность | REAL | дни | 0 – 365 | Время сохранения свойств аналита |
| `HL` | Период полураспада | REAL | часы | 0 – 8760 | Время, за которое концентрация уменьшается вдвое |
| `PC` | Энергопотребление | REAL | мВт | 0 – 1000 | Потребление энергии при детектировании |

**Ограничения валидации (`field_constraints['analyte']`):**

## 3.3 Иммобилизационный слой (Immobilization Layer)

Слой, обеспечивающий закрепление биораспознающих элементов на поверхности мемристивного слоя.

**Таблица в БД:** `ImmobilizationLayers`


| Обозначение | Название характеристики | Тип данных | Единицы измерения | Диапазон значений | Описание |
| :-- | :-- | :-- | :-- | :-- | :-- |
| `IM_ID` | Идентификатор слоя | VARCHAR | — | — | Уникальный ID (например, IM001) |
| `IM_Name` | Название слоя | VARCHAR | — | — | Тип иммобилизационного материала |
| `PH_Min` | pH минимум | REAL | единицы pH | 2.0 – 10.0 | Минимальный pH для стабильности материала |
| `PH_Max` | pH максимум | REAL | единицы pH | 2.0 – 10.0 | Максимальный pH для стабильности материала |
| `T_Min` | Температура минимум | REAL | °C | 4 – 120 | Минимальная рабочая температура |

| `TR` | Время отклика | REAL | секунды | 0 – 3600 | Время установления контакта с биослоем |

| `HL` | Долговечность | REAL | часы | 0 – 8760 | Ресурс работы без деградации |

```python
'durability': {'min': 0, 'max': 8760}
'power_consumption': {'min': 0, 'max': 1000}
```

**Особенность параметра MP (Модуль Юнга)**

Модуль Юнга (`MP` в базе данных, `young_modulus` в интерфейсе) — ключевой параметр механической совместимости слоев. Он показывает жесткость материала при деформации и измеряется в гигапаскалях (ГПа). Высокое значение модуля Юнга указывает на жесткий материал, низкое — на эластичный. Для оптимальной работы биосенсора важно, чтобы модули Юнга иммобилизационного и мемристивного слоев были сопоставимы для минимизации механических напряжений на границе раздела.

## 3.4 Мемристивный слой (Memristive Layer)

Транзисторный элемент с памятью сопротивления, преобразующий биохимический сигнал в электрический.

**Таблица в БД:** `MemristiveLayers`


| Обозначение | Название характеристики | Тип данных | Единицы измерения | Диапазон значений | Описание |
| :-- | :-- | :-- | :-- | :-- | :-- |
| `MEM_ID` | Идентификатор мемристора | VARCHAR | — | — | Уникальный ID (например, MEM001) |
| `MEM_Name` | Название мемристора | VARCHAR | — | — | Тип мемристивного материала |
| `PH_Min` | pH минимум | REAL | единицы pH | 2.0 – 10.0 | Минимальный pH для работы устройства |
| `PH_Max` | pH максимум | REAL | единицы pH | 2.0 – 10.0 | Максимальный pH для работы устройства |
| `T_Min` | Температура минимум | REAL | °C | 5 – 120 | Минимальная рабочая температура |
| `T_Max` | Температура максимум | REAL | °C | 5 – 120 | Максимальная рабочая температура |
| **`MP`** | **Модуль Юнга** | **REAL** | **ГПа** | **0 – 1000** | **Механическая жесткость мемристивного материала** |

| `TR` | Время отклика | REAL | секунды | 0 – 3600 | Время изменения сопротивления при детектировании |
