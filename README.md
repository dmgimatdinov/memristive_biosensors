
# Memristive Biosensors

Проект для работы с паспортами (данными характеристик) мемристивных биосенсоров.

## Установка

```bash
git clone https://github.com/dmgimatdinov/memristive_biosensors
cd memristive_biosensors
pip install -r requirements.txt
streamplit run app.py
```


## Использование

Загружайте файлы паспортов сенсоров для анализа и обработки данных. Поддержка SQL для баз данных характеристик.

## Описание

Мемристивные биосенсоры применяются в электрохимическом детектировании (например, PSA), с функционализацией антителами.
